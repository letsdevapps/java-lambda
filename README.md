# Java Lambda

Projeto generico para demonstração de funções Lambda do Java, tipos:
* Runnable: Função que não retorna valor
* Supplier: Função que retorna um valor, mas não recebe parâmetros
* Consumer: Função que recebe um valor, mas não retorna nada
* Function: Função que recebe um valor e retorna outro
* Predicate: Função que retorna um valor booleano

## Maven Generate

    mvn archetype:generate -DgroupId=com.pro -DartifactId=java-lambda -DarchetypeArtifactId=maven-archetype-quickstart -DinteractiveMode=false

    mvn clean package
